Les commandes clavier Ubuntu

Lanceur
"Alt+F1" = Ouvre le mode navigation au clavier du lanceur. "Super + Tabulation" = Bascule entre les applications via le lancer. "Super + 1 à 9" = Comme en cliquant sur une icône du lanceur. "Super + maj + 1 à 9" = Ouvre une nuovelle fenêtre dans l'application. "Super + T" = Ouvre la corbeille.
Affichage en tête de menu
"Alt (Appui court)" = Ouvre l'affichage tête haute. "Alt (Appui long)" = Affiche le menu des applications. "Alt + F10" = Ouvre le menu des indicateurs. "Fléche gauche ou droite" = Navigue entre les indicateur. "Impr" = prendre une capture d'écran. "Alt + Impr" = Prendre un capture d'écran de la fenêtre actuelle.
Tableau de bord
"Super (appui court)" = Ouvre l'acceuil du tableau de bord. "Super + A" = Ouvre la lentille *Applications* du tableau de bord. "Super + F" = Ouvre la lentille *Fichiers et dossiers* du tableau de bord. "Super + M" = Ouvre la lentille *Musique* du tableau de bord. "Super + C" = Ouvre la lentille *Photos* du tableau de bord. "Super + V" = Ouvre la lentille *Vidéos* du tableau de bord. "Ctrl + Tab" = Bascule entre les lentilles.
Fenêtres
"Super + W" = Déploie toutes les fenétres. "Ctrl + Super + W" = Déploie toutes les fenétres de l'application active. "Ctrl + Super + D" = Minimise toutes les fenêtres. "Ctrl + Super + Haut" = Maximise la fenétre actuelle. "Ctrl + Super + Bas" = Restaure ou minimise la fenétre courante. "Ctrl + Super + Gauche ou Droit" = Maximise la fenétre courante sur un demi-écran. "Alt + F4" = Ferme la fenétre courante.
