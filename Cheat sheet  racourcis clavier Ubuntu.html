Les commandes clavier Ubuntu

Lanceur
"alt+F1" =Ouvre le mode navigation au clavier du lanceur. "super + Tabulation" = Bascule entre les applications via le lancer. "super + 1 à 9" = Comme en cliquant sur une icône du lanceur. "super + maj + 1 à 9" = Ouvre une nuovelle fenêtre dans l'application. "super + T" = Ouvre la corbeille.
