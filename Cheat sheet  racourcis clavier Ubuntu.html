Les commandes clavier Ubuntu

Lanceur
"alt+F1" = Ouvre le mode navigation au clavier du lanceur. "super + Tabulation" = Bascule entre les applications via le lancer. "super + 1 à 9" = Comme en cliquant sur une icône du lanceur. "super + maj + 1 à 9" = Ouvre une nuovelle fenêtre dans l'application. "super + T" = Ouvre la corbeille.
Affichage en tête de menu
"alt (Appui court)" = Ouvre l'affichage tête haute. "alt (Appui long)" = Affiche le menu des applications. "alt + F10" = Ouvre le menu des indicateurs. "Fléche gauche ou droite" = Navigue entre les indicateur. "impr" = prendre une capture d'écran. "Alt + Impr" = Prendre un capture d'écran de la fenêtre actuelle.
